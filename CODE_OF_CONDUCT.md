# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## In short

Be respectful. No harassment, no discrimination, no personal attacks. Disagree about code, not about people.

## Scope

This applies in all project spaces — issues, pull requests, discussions, and any other interaction connected to the project.

## Reporting

If you experience or witness unacceptable behavior, contact **rtorcato@me.com**. All reports are reviewed and investigated and will result in a response that's deemed necessary and appropriate. Privacy of the reporter will be respected.
